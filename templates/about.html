About This Project

Deep learning for biometric blood group classification

Project Overview

This system uses Convolutional Neural Networks (CNN) combined with Grad-CAM (Gradient-weighted Class Activation Mapping) to predict human blood groups from fingerprint images. The model analyzes unique fingerprint patterns and provides visual explanations of its decision-making process, making the AI transparent and interpretable.
